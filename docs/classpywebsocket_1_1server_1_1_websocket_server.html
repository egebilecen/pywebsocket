◆ _encode_data()

bytes pywebsocket.server.WebsocketServer._encode_data ( bytes  data,
int  opcode 
)
staticprivate

Encodes the data that will be sent to client according to websocket packet structure and rules.

Parameters
dataData that will be sent to client.
opcodeOPCODE of frame.
Note
If OPCODE set to FrameType.TEXT_FRAME, client will receive data as UTF-8 string. If OPCODE set to FrameType.BINARY_FRAME, client will receive data as byte array.
Warning
- Raises exceptions.DATA_LENGTH_ERROR exception if data's length is bigger than 0xFFFFFFFFFFFFFFFF.
- Do not forget that all control frames MUST have a payload length of 125 bytes or less and MUST NOT be fragmented.
