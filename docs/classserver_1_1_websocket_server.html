staticprivate

Encodes the data that will be sent to client according to websocket packet structure and rules.

Parameters
dataData that will be sent to client.
frame_typeType of frame. It can only be FrameType.TEXT_FRAME or FrameType.BINARY_FRAME. If data sent as a FrameType.TEXT_FRAME, client will receive data as UTF-8 string. If data sent as a FrameType.BINARY_FRAME, client will receive data as byte array.
opcode_ovrOPCODE override. OPCODE will be set to this value if value is not None.
